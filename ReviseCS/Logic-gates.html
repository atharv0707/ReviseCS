Logic Gates

Truth Tables

Constructing truth tables is fundamental in understanding the behavior of different logic gates:

  • NOT Gate: Negates the input.
  • AND Gate: Outputs true only if all inputs are true.
  • OR Gate: Outputs true if at least one input is true.
  • XOR Gate: Outputs true if exactly one input is true.
  • NAND Gate: Outputs false only if all inputs are true.
  • NOR Gate: Outputs false if at least one input is true.
